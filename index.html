⭐ Make.com + LINE Messaging API (แนะนำ — ฟรี)
LINE Notify ถูกยกเลิกแล้ว — ใช้ LINE Messaging API แทน

ขั้นตอนการตั้งค่า (ทำครั้งเดียว ~15 นาที):
1. สมัคร make.com (ฟรี 1,000 ops/เดือน)
2. สร้าง Scenario: Webhooks → HTTP Request → LINE
3. HTTP URL: https://api.line.me/v2/bot/message/push
4. Header: Authorization: Bearer {Channel Access Token}
5. Copy Webhook URL ของ Make.com มาวางด้านล่าง

📌 DAV จะส่งคู่มือตั้งค่า Make.com + LINE ให้แยกต่างหาก
Webhook นี้รับ JSON: {"message": "..."} แล้วส่งต่อไป LINE
ถ้าว่างจะส่งตาม Webhook ที่ตั้งค่าใน Make.com
